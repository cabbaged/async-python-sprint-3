Примеры работы с мессенджером:

1. Запуск сервера
$ python run_server.py

2. Запуск клиента с юзернеймом UserA
python run_client.py --username UserA

Любой клиент при запуске настроен на запись в общий чат (general).
Любой клиент будет получать сообщения из чата general и приватные сообщения.

3. Клиент можно настроить на запись в приватный чат с каким-нибудь юзером, прописав например:
/switch_chat UserB

UserB будет получать сообщения от UserA в приватном чате, чтобы ответить в приватный чат ему нужно переключиться на него:
/switch_chat UserA

4. Вернуться в общий чат:
/switch_chat general


5. Отправка файла. Необходимо прописать "upload" и нажать Enter. Далее прописать полный путь до файла:
upload
/path/to/your/file

Получатели получат сообщение с id загруженного файла.
Файл, отправленный в чат general доступен всем пользователям.
Чат отправленный в приватном чате доступен только получателю.

6. Получение файла. !Необходимо перейти в чат, в котором файл был отправлен. Далее прописать download.
Далее прописать id желаемого файла и через пробел путь, куда его сохранить:
/switch_chat general
download
613d257f-7035-44c9-b635-220ff8edad06 ./myfile
